## 사용 방법
> python main.py <FEED_FILE\> <OUTPUT_DIR\>

<FEED_FILE\>에는 우편등기번호가 기재된 CSV파일을 입력합니다.

<OUTPUT_DIR\>에는 출력 결과물이 저장될 폴더명을 입력합니다.
